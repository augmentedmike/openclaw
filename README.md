# @miniclaw_official/openclaw

[![npm](https://img.shields.io/npm/v/@miniclaw_official/openclaw?style=for-the-badge)](https://www.npmjs.com/package/@miniclaw_official/openclaw)
[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg?style=for-the-badge)](LICENSE)

The AI agent engine that powers [MiniClaw](https://miniclaw.bot).

This is a fork of [OpenClaw](https://github.com/openclaw/openclaw) with MiniClaw-specific patches:

- Root-alias plugin loading
- `mc-*` plugin security whitelist

## Installation

You do **not** install this package directly. It is installed automatically by the MiniClaw installer.

**To get started with MiniClaw, visit [miniclaw.bot/install](https://miniclaw.bot/install).**
